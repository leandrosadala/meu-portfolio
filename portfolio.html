Automação de Processos de Gestão de Dados Geofísicos por Agentes de IA

Este é um projeto em andamento e seu principal objetivo é reduzir ou até eliminar a execução humana de tarefas repetitivas no processo de gestão de dados geofísicos, gerando economia e realocação das pessoas para a realização de atividades mais cognitivas. Para isso criamos agentes de IA e ferramentas computacionais para que sejam utilizadas por eles.

Certificadores de Dados Sísmicos

Este é um projeto em andamento e seu principal objetivo é certificar dados sísmicos adquiridos e processados para a exploração de petróleo, garantindo a consistência, integridade e validade dos dados de acordo com um padrão de qualidade pré-definido. Para isso, ferramentas computacionais de validação são desenvolvidas para checagem das informações (metadados e dados propriamente ditos) contidas nos arquivos de dados.

Imageamento Sísmico via Marchenko

O objetivo deste projeto foi criar uma solução computacional alternativa às convencionais para o imageamento de dados sísmicos na exploração de petróleo. A principal vantagem dele sobre os demais é a capacidade de geração de imagens da subsuperfície com maior resolução, evitando procedimentos adicionais de elevado custo computacional e que nem sempre produzem resultados relevantes.

Atenuação de Múltiplas Internas via Marchenko

O objetivo deste projeto foi a criação de uma ferramenta computacional para a atenuação ou remoção de múltiplas internas responsáveis pela diminuição da resolução das imagens sísmicas utilizadas na interpretação de feições geológicas em subsuperfície indicativas da presença de petróleo. A sua utilização evita a utilização de filtros antes e após o imageamento sísmico, reduzindo custos na etapa de processamento sísmico.

Atenuação de ruídos "fantasmas" em dados sísmicos do pré-sal

Objetivo deste projeto foi o desenvolvimento de uma ferramento computacional para a atenuação de ruídos presentes em dados sísmicos de exploração de petróleo em anbientes marítimos, denominados "fantasmas". Recebem este nome por se comportarem como um sinal sísmico secundário semelhante a fonte principal e que é registrada logo após poucos milisegundos do sinal principal, ocasionando queda de resolução das imagens sísmicas posteriormente geradas.

Redatumação por Ponto de Focalização Comum

O objetivo deste projeto foi a criação de uma ferramenta computacional para a alteração do datum original dos dados sísmicos levantados visando o aumento da resolução das imagens sísmicas da subsuperfície terrestre, principalmente, para a prospecção de petróleo. A redatumação funciona como a focalização de uma câmera fotográfica: ao ajustar o foco da lente com a finalidade de aproximação do alvo, o operador da câmera verá mais nitidamente o que deseja. O mesmo ocorre com os dados sísmicos, redatumando (simulando) os dados adquiridos num datum mais próximo da profundidade do alvo petrolífero, mais facil de identificá-lo.
